ansible-tools
=============

Vagrant box and a few scripts to test ansible features.

Installation
----
```console
$ git clone git://github.com/franlr/ansible-tools
$ cd ansible-tools
$ vagrant up --provider virtualbox
```

Supported OS
----
* Debian: 7,8
* CentOS: 6
* OpenSuSE 11
* Ubuntu: 12.04, 14.04
